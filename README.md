# Pokédex React

Aplicación web de Pokédex desarrollada con React que consume la PokeAPI.

## Tecnologías

- React 18
- React Router DOM
- React Hook Form
- CSS (sin librerías)
- PokeAPI

## Funcionalidades

- Listado de los 151 Pokémon originales
- Búsqueda por nombre en tiempo real
- Filtro por tipo
- Página de detalle con estadísticas
- Sistema de favoritos con persistencia en localStorage
- Formulario de contacto con validaciones
- Diseño responsive

## Instalación
```bash
npm install
npm run dev
```

## Estructura del proyecto
```
src/
├── components/    # Componentes reutilizables
├── pages/         # Páginas de la aplicación
├── hooks/         # Custom hooks
├── context/       # Context API
├── services/      # Llamadas a la API
└── utils/         # Funciones auxiliares
```
